DataProvider¶

class DataProvider.dataprovider.NetworkDataProvider(url=None)¶

The DataService class provides a base class for any remote service such as oData or soap.

setBasicAuth(username, password)¶

Once set, every request sent to the server will append a base-64 encoded basicAuth header encoded with the provided username and password.

wait()¶

Block until the dataprovider has no pending requests
